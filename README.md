# NativeScript Tappy

{N} plugin for integrating with the iOS sdk for [https://github.com/TapTrack/TCMPTappy-iOS]()


## Demo

The demo app scans for TapTrack devices, connects, and performs read and write actions.
To run the demo:

cd demo
npm install
tns platform add ios
tns run ios
